# Music store

Costruire un negozio di musica

Funzionalita:
- Si può aggiungere album
  - ha un artista
  - ha un genere
  - ha un prezzo
- Si può gestire gli artisti
  - aggiungere, cancellare o modificare
- Si può aggiungere un genere
- Si può Filtrare per genere o per artista o vedere tutti gli album presenti
- Si può aggiungere/rimuovere un album al carrello
- Si può acquistare il carrello

Bonus:
- Provare ad astrarre le liste in un oggetto che le adatta esponendo solo i metodi semplici CRUD
  - esempio: il carrello invece di avere una lista di oggetti, ah un oggetto CartRespository che continene la lista, incapsulando come viene salvata la lista
